# RPG Level Mod (Minecraft Forge 1.20.1)

A Minecraft Forge mod for version 1.20.1 that adds RPG-like progression systems and gameplay enhancements.

## Features
- Level progression system
- RPG-style mechanics
- Gameplay improvements for survival experience

## Requirements
- Minecraft 1.20.1
- Forge 47.x
- Java 17+

## Installation
1. Install Minecraft Forge 1.20.1
2. Download the mod `.jar`
3. Place it in your `mods/` folder
4. Launch the game
